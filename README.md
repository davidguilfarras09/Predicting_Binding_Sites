# Predicting_Binding_Sites
This project leverages ESM2, a state-of-the-art protein language model developed by Meta AI, to generate residue-level embeddings from raw protein sequences. These embeddings are then used as input features for machine learning models trained to predict binding sites,residues that interact with ligands, other proteins, or small molecules. The goal is to accurately identify functional regions of proteins using sequence information alone, without requiring experimental 3D structures.
